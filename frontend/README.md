### Sample-app frontend

NOTE: You don't need to run any of this if you have read `sample-app/README.md` and work inside `minikube`. 

#### Install

```
npm i
```

#### Local react server and live reload on your host machine

```
npm start
```

#### Lint and flow before commit

```
npm run check-all
```

### Production build

```
npm run build
```
